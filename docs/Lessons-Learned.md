# Lessons Learned

## Project Summary

This project provided practical experience building and operating a Security Operations Center environment from the ground up.

The lab simulated realistic security monitoring and investigation workflows used by enterprise SOC teams.

---

## Technical Skills Gained

### SIEM Administration

Learned to:

* Deploy Wazuh
* Configure log collection
* Monitor endpoint activity
* Investigate alerts

---

### Endpoint Monitoring

Learned to:

* Install Sysmon
* Configure event collection
* Analyze Windows logs
* Monitor process activity

---

### Threat Hunting

Learned to:

* Search security telemetry
* Investigate suspicious activity
* Correlate multiple events
* Validate detections

---

### MITRE ATT&CK Mapping

Learned to:

* Associate alerts with ATT&CK techniques

* Improve investigation workflows

### Incident Response

Learned to:

* Triage alerts
* Investigate incidents
* Analyze evidence
* Document findings
