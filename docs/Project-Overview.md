# Project Overview

## Project Title

SOC Home Lab: Threat Detection and Incident Response Using Wazuh SIEM

---

## Executive Summary

This project demonstrates the deployment and operation of a Security Operations Center (SOC) Home Lab using Wazuh SIEM, Windows 10, Ubuntu Server, Kali Linux, and Sysmon.

The objective was to simulate enterprise-grade security monitoring, collect endpoint telemetry, detect suspicious activities, investigate alerts, and perform threat hunting using industry-standard defensive security tools.

The environment was designed to replicate a real-world SOC workflow where security analysts continuously monitor systems, analyze alerts, investigate incidents, and map attacker techniques to the MITRE ATT&CK Framework.

---

## Business Objective

Organizations require continuous monitoring of endpoints and network activity to detect threats before they impact business operations.

This project addresses that requirement by implementing:

* Centralized log collection
* Endpoint monitoring
* Security event detection
* Threat hunting
* Incident investigation
* ATT&CK-based threat analysis
